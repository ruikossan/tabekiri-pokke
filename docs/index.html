たべきりポッケ

冷蔵庫・冷凍庫・食品棚の中身をまとめて管理。賞味期限切れ、買い忘れ、同じ食品の買いすぎを減らすための食品ストックアプリです。

サポートを見る プライバシーポリシー

できること

家にある食品を、買い物前にも調理前にも確認しやすく。

食品名、数量、保管場所、期限、写真、バーコード番号を記録できます。登録データは端末内に保存され、ログインなしで利用できます。

食品ストック管理

冷蔵庫、冷凍庫、常温、食品棚など、保管場所ごとに食品を整理できます。

期限チェック

期限切れ、7日以内、30日以内の食品を確認し、食べきりや補充の判断に役立てられます。

買い物リスト

不足しそうな食品や、よく買う食品を買い物リストにまとめられます。

主な機能

毎日の食品管理に必要な操作をひとまとめに。

  • 食品の登録、編集、削除
  • 数量、単位、カテゴリ、保管場所、賞味期限・消費期限、メモの管理
  • 食品写真の登録とバーコード番号の保存
  • 使った、買った、確認した履歴の記録
  • 期限やチェック日のローカル通知
  • お試しデータの投入と初期状態へのリセット

データの扱い

登録内容は端末内に保存されます。

現在の実装では、アカウント登録、クラウド同期、外部サーバーへの食品データ送信、広告SDK、分析SDKはありません。詳しくはプライバシーポリシーをご確認ください。
